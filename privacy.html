Privacy Policy

App name: Fanvue TikTok Automation

Last updated: May 13, 2026

Overview

Fanvue TikTok Automation ("the App") is a content management platform that enables content creators to automate video publishing to TikTok via TikTok's official Content Posting API. This Privacy Policy explains how the App handles data collected through TikTok's OAuth authorization flow.

Data We Collect

Fanvue TikTok Automation collects only the following data, obtained through TikTok's OAuth authorization flow:

  • TikTok User ID (open_id) — used solely to identify the authorized creator account for posting
  • TikTok access token and refresh token — stored securely and used only to authenticate API requests on behalf of the creator
  • Basic profile information (user.info.basic scope) — used to confirm account identity

We do not collect names, email addresses, follower data, video analytics, or any other personal information beyond what is required to authenticate and post content.

How We Use Data

Data collected by Fanvue TikTok Automation is used exclusively to publish video content to the authorized TikTok creator account. It is never shared with third parties, sold, or used for advertising, analytics, or any purpose other than content posting.

Data Storage

All authentication tokens are stored securely on the creator's own device or server. Fanvue TikTok Automation does not transmit data to external servers beyond what is required to communicate with TikTok's official API.

Third-Party Services

Fanvue TikTok Automation communicates only with TikTok's official API (open.tiktokapis.com). No other third-party services receive user data.

User Rights

Creators may revoke Fanvue TikTok Automation's access to their TikTok account at any time through TikTok's Connected Apps settings. Upon revocation, all stored tokens are invalidated.

Children's Privacy

Fanvue TikTok Automation is not intended for use by individuals under the age of 18 and does not knowingly collect data from minors.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected by updating the "Last updated" date above.
